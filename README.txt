# 🍼 Dulces Sueños - Sistema de Gestión de Inventario

¡Bienvenido a **Dulces Sueños**! Esta es una aplicación Full Stack diseñada para una boutique de bebés. Permite gestionar productos, categorías, precios y stock de manera eficiente, con una interfaz tierna y profesional.

## 🚀 Características Principales
- **CRUD Completo:** Registro, consulta, edición y eliminación de productos.
- **Dashboard Inteligente:** Cálculo en tiempo real del valor total del inventario y alertas de stock bajo.
- **Filtros Dinámicos:** Búsqueda por categorías que se actualizan automáticamente.
- **Exportación Pro:** Generación de reportes detallados en formato **Microsoft Excel**.
- **UX/UI Moderna:** Alertas estéticas con SweetAlert2 y diseño responsivo inspirado en una paleta de rosas.

## 🛠️ Tecnologías Utilizadas
- **Backend:** Java 17, Spring Boot, Spring Data JPA.
- **Base de Datos:** MySQL / MariaDB.
- **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (Vanilla).
- **Librerías:** SweetAlert2 (Alertas), SheetJS (Excel).

## 📋 Requisitos e Instalación
1. **Base de Datos:** - Importa el archivo `database_dulces_suenos.sql` en tu servidor MySQL (puedes usar HeidiSQL o MySQL Workbench).
   - Configura tus credenciales en el archivo `application.properties` del proyecto Java.
2. **Backend:** - Ejecuta el proyecto Spring Boot desde tu IDE favorito (IntelliJ, Eclipse, VS Code).
3. **Frontend:**
   - Abre el archivo `index.html` en tu navegador o usa una extensión como Live Server.
